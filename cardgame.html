Juego de Cartas

Juego de cartas utilizando Javascript, HTML y CSS.

Juego con tematica horror para Halloween 2021.

El mismo esta inspirado en el anime (animacion japonesa) "Serial Experiments Lain" se basa en encontar las cartas pares en menos de 100 segundos. Las animaciones estan hechas en CSS sobre una estructura en HTML y solo se uso Javascript para programar el funcionamiento del juego.
